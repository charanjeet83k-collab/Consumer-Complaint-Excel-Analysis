## Consumer Complaint Analysis (Excel)
Excel-based analysis of consumer complaint data to uncover trends, issue categories, and complaint distribution.

## 🎯 Project Objective

The objective of this project is to analyze consumer complaint data using Microsoft Excel to identify complaint patterns, customer issues, and trends across different products and companies.

The analysis aims to transform raw complaint data into meaningful insights that help understand the most common customer issues, complaint resolution patterns, and product categories generating the highest number of complaints.

## 📌 Project Overview

This project focuses on analyzing consumer complaint data using Excel tools such as data cleaning, pivot tables, filtering, and data visualization.

The dataset contains customer complaints related to financial products and services. By organizing and analyzing the dataset, the project identifies patterns in complaint types, product categories, company responses, and resolution timelines.

The analysis demonstrates how Excel can be used as a powerful tool for data exploration, trend analysis, and reporting.

## 🛠 Tech Stack

The analysis was performed using the following tools and techniques:

📊 **Microsoft Excel**
Used for data cleaning, analysis, and visualization.

📑 **Pivot Tables**
Used to summarize complaint data and analyze complaint distribution across products and companies.

📉 **Charts & Visualizations**
Used to present complaint trends and category distributions.

🧹 **Data Cleaning**
Handled missing values, standardized data fields, and structured the dataset for analysis.

🗂 **GitHub**
Used to host and document the project as part of a professional data analytics portfolio.

## 📂 Dataset Description

The dataset contains records of consumer complaints submitted against different financial products and companies.

Key data fields include:
- Complaint ID
- Product Category
- Issue Type
- Company Name
- State / Location
- Complaint Status
- Date Received
- Date Sent to Company

This dataset helps analyze complaint patterns, company responses, and product-related issues.

## 📊 Key Analysis Performed

Several analyses were conducted to better understand complaint trends:

📌**Product Category Analysis**
Identified which financial products receive the highest number of consumer complaints.

📌**Issue Type Analysis**
Analyzed the most common types of complaints reported by customers.

📌**Company Complaint Distribution**
Evaluated which companies received the highest number of complaints.

📌**Complaint Trends Over Time**
Examined complaint patterns based on dates to understand changes in complaint volume.

📌**Complaint Resolution Status**
Analyzed the distribution of resolved, pending, and disputed complaints.

## 📈 Insights Generated

Some key insights derived from the analysis include:

- Identification of products generating the highest number of complaints
- Understanding the most common customer issues across products
- Analysis of complaint distribution across companies
- Trends in complaint volume over time
- Overview of complaint resolution patterns

These insights help organizations understand customer concerns and improve service quality.
